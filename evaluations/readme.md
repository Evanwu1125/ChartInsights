We conducted a total of four sets of evaluations on GPT-4V. The first set of evaluations was a general assessment of GPT-4V's performance,
utilizing the test dataset from ChartInsights. 
The second set of evaluations involved altering 15 types of chart syntax. 
The third set of evaluations introduced noise into the images. 
The fourth set of evaluations incorporated the addition of visual prompts and the design of Chain-of-Charts to the images.
